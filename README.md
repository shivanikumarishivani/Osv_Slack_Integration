# Clarify Task Details API

## Overview
The **Clarify Task Details API** is a RESTful API built using **Node.js** and **Express.js**.  
It allows users to manage tasks with operations like creating, updating, retrieving, and deleting tasks. This project is **backend-only**, designed to be integrated with any frontend or used via API clients like Postman.

## Features
- Get all tasks
- Get a single task by ID
- Create a new task
- Update an existing task
- Delete a task
- JSON-based request and response format

## Technologies
- Node.js
- Express.js
- Body-parser
- Nodemon (for development)
- (Optional) CORS for cross-origin requests
